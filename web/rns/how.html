🔎 Search for a name

Open the app and type the name you want — say ravi. It instantly tells you whether ravi.rama is available. Try a few until you find the one you love.

📝 Register it

Choose how long you want it — 1 to 5 years — and confirm one transaction in your wallet. The name is now registered to your address, on-chain and verifiable.

Flat 100 RAMA per year, any length
3

🎯 Point it at your address

Set which wallet the name resolves to (it defaults to yours). From now on, anyone sending to ravi.rama reaches that address. You can change the target any time.

Add profile records too — avatar, URL, socials
4

🔁 Manage & renew

From the app you can update the address, add sub-names like pay.ravi.rama, transfer the name to someone else, or renew before it expires to keep it forever.
